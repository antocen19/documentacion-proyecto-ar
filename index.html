Proyecto de Análisis de Sistemas: Explorador de Hardware con Realidad Aumentada Web

Documentación de la solución de visualización inmersiva.

Iniciar Experiencia de RA

1. Análisis del Problema y Justificación del Sistema

    El problema identificado es la limitación de la visualización 2D en la enseñanza del hardware (CPU, GPU, RAM), que no permite entender correctamente su escala y ubicación espacial.
    La solución es un sistema WebAR (Realidad Aumentada Web). Elegimos esta tecnología por su accesibilidad (no requiere instalar apps) y su capacidad para ofrecer una visualización inmersiva 3D directamente sobre un marcador físico. Este enfoque transforma un concepto abstracto en una experiencia de aprendizaje tangible y eficiente.

2. Diseño y Componentes de la Solución

    El diseño se basa en el seguimiento de marcadores (marker tracking) para la activación de modelos 3D. Cada marcador físico está asociado a un componente de PC:

Marcador 1 (ej. Hiro): Activa el modelo CPU.

Marcador 2 (ej. Kaije): Activa el modelo GPU.

Marcador 3 (ej. 'A'): Activa el modelo RAM.

El sistema garantiza que solo el modelo 3D correspondiente al marcador detectado sea renderizado, manteniendo la experiencia clara y rápida.

3. Arquitectura Técnica y Stack de Desarrollo

    El proyecto utiliza un stack de código abierto para su desarrollo y despliegue:

Librería Principal: MindAR: Es la librería de JavaScript ligera y de alto rendimiento que se encarga del reconocimiento del marcador y el cálculo de la posición de la cámara (WebXR).

Contenido 3D: Modelos optimizados en formato GLTF/GLB.

Plataforma de Despliegue: GitHub Pages: Se usa para el hosting estático, lo cual es crucial porque garantiza el protocolo HTTPS. El HTTPS es un requisito de seguridad fundamental para que los navegadores permitan el acceso a la cámara y, por lo tanto, para que la RA funcione.

Herramientas de Edición: El código se gestionó y editó usando Visual Studio Code.

4. Guía de Uso y Acceso

  1. Abre esta página en tu dispositivo móvil.
  2. Presiona el botón "Iniciar Experiencia de RA".
  3. Escanea los marcadores impresos.
